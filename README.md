# Mini Flipdot Display

This project is a mini flipdot display to display the time/weather/drawings/etc. This repo contains the array PCB, test PCB, controller PCB, software for the controller, and the case. For more details see my [post on my website](https://tylerebowers.com/posts/mini_flipdots).
